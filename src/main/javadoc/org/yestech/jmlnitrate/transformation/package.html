Functionality

The JMLNitrate Transformation package provides Transformation Engines that can dynamically take either an inbound request or outbound response for information and transform the raw information into a uniform request or response that the microkernel can understand. In the case of the inbound the Transformer takes the raw request and transforms it to a unfirom request. The outbound takes a uniform response and transforms it to a raw response.
